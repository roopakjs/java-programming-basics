# Java Programming Basics

This repository contains basic Java programs created for learning core programming concepts and improving logical thinking.

## Programs Included
- HelloWorld.java – Prints a simple message
- Fibonacci.java – Generates Fibonacci sequence
- Palindrome.java – Checks whether a number is a palindrome
- PrimeNumber.java – Checks whether a number is prime
- Factorial.java – Calculates factorial of a number
- ReverseNumber.java – Reverses a given number

## Concepts Covered
- Loops (for, while)
- Conditional statements (if-else)
- Number logic
- Basic algorithms

## Technologies Used
- Java

## Purpose
This repository is intended for beginners who are learning Java and want to strengthen their programming fundamentals.
